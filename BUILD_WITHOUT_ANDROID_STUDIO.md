# Build dietie without Android Studio

1. Upload this project folder to a GitHub repository.
2. Open the repository's **Actions** tab.
3. Select **Build dietie APK**.
4. Click **Run workflow**.
5. When the run turns green, open it and download the **dietie-debug-apk** artifact.
6. Extract the downloaded artifact zip. The APK is inside it.

This workflow uses stable Android tooling and installs Android SDK 36, so it should not need preview SDK packages.
